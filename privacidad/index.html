Privacidad y tratamiento de datos

Lineamientos para el uso de formularios y comunicación institucional.

Uso de información

Los datos enviados mediante formularios se usan para responder solicitudes, orientar evaluaciones preliminares y mantener trazabilidad de contacto.

Confidencialidad

La información técnica será tratada con reserva institucional. Para intercambios detallados con terceros, se recomienda formalizar acuerdos de confidencialidad.

Solicitud de eliminación o corrección

Escríbenos a OTL@UNAB.CL para actualizar o eliminar datos de contacto enviados por este sitio.
